# LUNA-WORLD

루나의 두 번째 마을. **LLM·API 호출 0, 운영 비용 0** — 순수 규칙 코드만으로 주민들이
실시간으로 살아가는 브라우저 시뮬레이션 실험이다. 루나의 도서관 AI마을이 "말의 창발"을
관찰했다면, 여기서는 "구조의 창발"을 관찰한다.

- 형태: 단일 HTML(캔버스 지도 + 관찰 패널), 시뮬레이션 전체가 브라우저에서 구동
- 배포: Cloud Run 정적 서빙 (최소 인스턴스 **0 유지** — 운영 수칙)
- 설치형 웹앱(PWA): manifest + 서비스 워커 + 자체 SVG 로고 — 홈 화면에 설치되고
  오프라인에서도 돈다 (상태는 localStorage에 있으므로 껍데기 캐시만으로 완전 동작)
- 상태: 폐기됐던 실시간 프로토타입(`index.html`)을 부활시켜 출발점으로 삼음 (2026-07-19)

## 프로토타입이 이미 갖춘 것

- Generative Agents식 기억 회상(최근성·중요도·관련성)과 성찰(기억 묶음)
- 확신도를 갖고 사람을 타고 퍼지는 소식 + 관측대 검증 (Talk of the Town 계열)
- 하루 일정 계획 + 상황 반응, 프로젝트·기술 서랍, 입력 대기열
- 시드 기반 난수(재현 가능), 0.05초 고정 스텝(결정론), localStorage 저장
- 멈춤 / 한 걸음 / 배속 관찰 컨트롤
- 대화·사건 문장을 Tracery식 치환 문법으로 생성 + 한국어 조사 자동 처리
- The Sims식 광고 결정층 — 건물이 효용을 광고하고 주민이 점수화 (행동 점수표 카드)
- 따라잡기 — 탭이 닫힌 시간을 저해상도 규칙으로 메꿈, 최대 사흘치 (Veloren rtsim 패턴)
- 기억이 영구화될 때 성격 4축(낙관·신중·사교·끈기)이 이동하고 행동에 되먹임 (Dwarf Fortress 패턴)
- 사건 감독 3종(카산드라·피비·랜디) — 마을 상태를 읽고 사건만 투하, 위기 후 회복 보장 (RimWorld 패턴)
- 대화의 질감 묶음 — DF식 취향 테이블(무의미하게 구체적인 취향이 대화 주제·광장 글·거처
  흔적·모임 이름에 스며듦, 같은 취향의 발견이 관계를 당김), 대화가 회상 기억을 실제로 인용,
  주민별 말투 문법 오버라이드, Tracery 템플릿 증량
- 소문 왜곡 (Talk of the Town) — 말이 사람을 타며 판본이 미끄러지고(장소 교체·과장·꼬리),
  주민마다 자기가 들은 판본을 가지며, 글은 왜곡되지 않고, 관측대 확인이 판본을 교정
- 이야기 발굴 (TotT drama.py) — 연대기 화면이 사건 로그·관계·약속·소문에서
  "부딪히는 두 사람", "다시 이어진 매듭", "말이 걸어간 길" 같은 서사 패턴을 사후 검색
- **루나의 도서관 이식 6종** ([`docs/research/luna-library.md`](docs/research/luna-library.md)) —
  약속(대화에서 생겨 동선을 끌어당기고, 파기가 쌓이면 평판 소문·마을 관습이 됨),
  쌍 중력(호감 점수가 아니라 "다시 엮일 이유"의 매번 재계산 합), 기억-정합 소문(일치=증폭·
  충돌=역소문·무경험=첫인상), 욕망 되먹임(갈등→화해, 완성→자랑, 낡음→손질), 개인 거주지 +
  장소의 역사(굳은 기억이 방 안의 흔적으로), 광장 글(시간차 온라인 전파)과 주민 발의 행사
  (초대가 소문망을 타는 발렌타인 파티 문법)
