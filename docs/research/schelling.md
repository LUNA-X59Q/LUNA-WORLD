# Schelling 분리 모델 — 온건한 선호, 극단적 결과 (1971)

- **한 줄**: "이웃의 1/3만 나와 같으면 만족"이라는 관용적 개인들이 모여도 극단적인 거주지 분리가 창발한다는, ABM 역사상 가장 유명한 반직관.
- **주체·시기**: Thomas Schelling, "Dynamic Models of Segregation" (1971). 이 연구 계열로 2005년 노벨 경제학상.
- **코드**: [mesa 공식 예제](https://github.com/mesa/mesa/tree/main/mesa/examples), NetLogo Segregation 모델 등 구현체 무수히 많음.

## 무엇인가

격자에 두 부류의 에이전트를 놓고 규칙 하나만 준다: "이웃 중 같은 부류의 비율이 임계값
미만이면 빈 칸으로 이사한다." 임계값이 30%처럼 낮아도(= 개인은 꽤 관용적이어도) 이사의
연쇄가 일어나 지도는 뚜렷하게 갈라진 구역들로 굳는다. **개인의 온건한 선호가 집단의
극단적 패턴을 만든다** — 미시 동기와 거시 결과는 다른 것이라는 ABM의 근본 교훈.

## 핵심 메커니즘

- 만족 판정(이웃 유사도 ≥ 임계값) → 불만족 시 무작위 빈 칸으로 이동 → 반복.
- 파라미터 단 하나(임계값)로 상전이가 보인다 — 관찰 도구로서 완벽한 최소 모델.

## LUNA-WORLD에 가져올 것

1. **거주지 동학** — 주민이 살 곳을 고르는 규칙(친한 이웃 선호 등)만 넣어도 마을에
   "구역"이 저절로 생긴다. 지도형 UI에서 가장 보기 좋은 창발.
2. **임계값 슬라이더** — 규칙 상수 하나를 실시간으로 바꾸며 상전이를 관찰하는 경험은
   이 장르 최고의 "아하 순간"이다. 관찰 UI 설계의 본보기.
3. **반직관의 가치** — 좋은 창발은 "규칙을 보면 당연한데 결과는 놀라운" 것. 마을 규칙을
   설계할 때의 품질 기준으로 삼을 것.

## 한계·주의

- 모델 자체는 5분이면 구현한다 — 가치는 코드가 아니라 관찰 설계에 있다.
