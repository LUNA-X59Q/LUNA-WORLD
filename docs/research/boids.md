# Boids — 규칙 3개로 군집이 난다 (1987)

- **한 줄**: 분리·정렬·응집 세 규칙만으로 새떼의 군집 비행을 재현한, "창발"이라는 개념의 대중적 원점.
- **주체·시기**: Craig Reynolds, "Flocks, Herds, and Schools: A Distributed Behavioral Model" (SIGGRAPH 1987).
- **코드**: [mesa 예제(boid_flockers)](https://github.com/mesa/mesa/tree/main/mesa/examples), 웹에 JS 구현 무수히 많음. 영화 VFX(배트맨 리턴즈의 박쥐떼)에도 쓰인 유서 깊은 알고리즘.

## 무엇인가

각 개체는 이웃만 보고 세 가지를 계산한다:
1. **분리(separation)** — 너무 가까우면 떨어진다
2. **정렬(alignment)** — 이웃의 평균 방향에 맞춘다
3. **응집(cohesion)** — 이웃의 무게중심 쪽으로 끈다

중앙 지휘 없이 이 셋의 가중 합만으로, 갈라졌다 합쳐지고 장애물을 감아 도는 "무리"가
나온다. 지역 규칙 → 전역 패턴의 가장 깨끗한 시연.

## LUNA-WORLD에 가져올 것

1. **군중 이동의 질감** — 주민들이 광장으로 모이거나 축제에서 흩어질 때 보이드 조향력을
   섞으면, 목표점 직선 이동(현재 프로토타입)보다 훨씬 "살아있는" 움직임이 된다.
   실시간 지도의 관찰 재미는 절반이 움직임의 질감이다.
2. **조향(steering) 어휘** — Reynolds의 후속 작업(seek/flee/wander/arrive…)은 실시간
   이동 연출의 표준 어휘다. 이동 코드를 이 어휘로 정리하면 확장이 쉽다.

## 한계·주의

- 사회성 없는 순수 운동 모델 — 우리에겐 연출 부품이지 뼈대가 아니다.
- 이웃 탐색이 O(n²)이므로 주민 수를 늘릴 땐 공간 해시 필요(수백 명까진 그냥도 됨).
