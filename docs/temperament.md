# 마을 기질 표기법 — 코드를 고치지 않고 마을을 편집한다 (2026-07-19)

> 로드맵 #11 (Kismet, [kismet.md](research/kismet.md)) 의 이식. 상단 헤더의 **기질 ✍️** 버튼으로
> 편집기를 열고, 아래 표기로 한 줄에 규칙 하나씩 적은 뒤 **적용**을 누른다.
> 기질은 이 기기(localStorage)에 남아 **새 마을을 만들어도 이어진다** — 세계 상태가 아니라
> 저작물이기 때문이다. Kismet의 교훈대로 저작(거의 자연어 규칙)과 실행(가중치 합산)을 분리했다.

## 표기 세 가지

`#`으로 시작하는 줄과 빈 줄은 무시된다. 알아듣지 못한 줄은 적용 시 행 번호와 함께 알려준다.

### ① 대화 수의 의지 기울이기

```
<수> 의지 +N
<수> 의지 -N
```

- 수: `화해` `전달` `조율` `취향` `회상` `갈등` `약속` `잡담` (`SOCIAL_RULES`의 move 이름)
- 대화가 일어날 때마다 계산되는 의지 점수에 N이 그대로 더해진다. 참고로 기본 의지는
  대략 10~60 사이를 오가고 무작위 가산이 ±10이라, ±5는 기울기, ±20은 지형 변화다.
- 예: `갈등 의지 -8` — 싸움이 드문 온화한 마을. `잡담 의지 +12` — 수다스러운 마을.
- 주민 패널의 "대화 의지 순위 — 왜 이 말을 했나"에서 효과를 눈으로 확인할 수 있다.

### ② 문장 풀에 보태기

```
문장 <풀>: <문장>
```

- 풀: `주제` `장소` `긍정` `신중` `맞장구` `마찰` `글감` `확인말` `추진말` `시험말` 등
  `G` 문법의 모든 심볼.
- `#주제.eun#` 같은 치환·조사 문법을 그대로 쓸 수 있다 — 적은 문장은 기본 풀과 합쳐져
  무반복 뽑기(`pickFresh`)에 함께 들어간다.
- 예: `문장 주제: 겨울 등불 축제` — 주민들이 겨울 축제 이야기를 하기 시작한다.

### ③ 관습 심기

```
관습: <문장>
관습(<키>): <문장>
```

- 키 없이 적으면 전시용 관습으로 "마을의 관습" 카드에 뜬다.
- 키가 `되물음` `화해` `모임` `약속`이면 시뮬레이션이 그 관습을 이미 가진 것처럼
  행동 확률에 되먹인다 (예: `관습(되물음): …` → 소문 왜곡 절반).
- 기질에서 심은 관습은 기질에서 지우면 함께 사라진다. 시뮬레이션이 스스로 키운
  관습은 건드리지 않는다.

## 예시 — 온화하고 수다스러운 축제 마을

```
# 다툼은 드물게, 화해는 빠르게
갈등 의지 -8
화해 의지 +10
잡담 의지 +6

# 계절 화제
문장 주제: 겨울 등불 축제
문장 주제: 첫눈 오는 날의 약속
문장 맞장구: 그 말, 오래 두고 생각할게.

# 자리를 여는 마을
관습(모임): 보름마다 무대에서 등불 저녁을 연다
```

## 구현 접점

| 표기 | 코드 접점 |
|---|---|
| 의지 | `rankSocialMoves()` — `TEMPER.moves[move]` 가산 |
| 문장 | `gen()` — `TEMPER.pools[sym]`를 `G[sym]`에 합류 |
| 관습 | `applyTemper()` — `state.customs`에 `temper:true`로 심고 갈아끼움 |
| 파서 | `parseTemper()` — 행 단위, 오류는 행 번호로 보고 |
| 보존 | localStorage `ai-village-temperament` — 세이브와 분리 |
